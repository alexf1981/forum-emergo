# Forum Emergo

**Forum Emergo** is a habit-tracking RPG that transforms your daily tasks into a quest for imperial glory. Built with React and designed for mobile-first usage.

## Features

- **Gamification**: Earn gold and army strength by completing habits (Virtues) and tasks (Mandata).
- **RPG Elements**: Level up heroes, find loot, and battle enemies.
- **Data Persistence**: All progress is saved locally in your browser.
- **Import/Export**: Backup your "Kronieken" (save data) to a JSON file.

## Technical Details

- **Framework**: React 18 (via unpkg/Babel Standalone)
- **Styling**: Native CSS3 with responsive design
- **Storage**: LocalStorage

## How to Run Locally

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. No build step or server required!

## Deployment

This project is static-ready.

### GitHub Pages
1. Go to Settings > Pages.
2. Select `main` branch as source.
3. Save.

### Netlify / Vercel
1. Drag and drop the folder into the deployment area.
